# piexifjs.d.ts

> piexifjs 1.x 类型定义

## 使用

```bash
pnpm add -D git+https://github.com/kohaiy/piexifjs.d.ts.git
```

`package.json`
```json
{
  "devDependencies": {
    "piexifjs.d.ts.git": "github:kohaiy/piexifjs.d.ts"
  }
}
```

`tsconfig.json`
```json
{
  "include": ["node_modules/piexifjs.d.ts.git/piexifjs.d.ts"]
}
```
